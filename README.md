# CEngine
## Rus
Консольный движок на Python - Рисуйте в консоли или создавайте Pixel-Арты с помощью символов.
(Создано на Python)

Пример куба: 

```
CEngine.draw("-", 10, "allup")
CEngine.draw("|          |", 1, None)
CEngine.draw("|          |", 1, None)
CEngine.draw("|          |", 1, None)
CEngine.draw("-", 10, "alldown")
```
